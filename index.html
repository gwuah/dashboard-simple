Savannah Zone Agricultural Productivity Improvement Project (SAZAPIP)

The Savannah Zone Agricultural Productivity Improvement Project (SAZAPIP) is set up to help solve some of the bottlenecks facing agricultural productivity (seeds, fertilizer), infrastructure deficits (irrigation, feeder roads) and policy gaps in Ghana, which if unaddressed would deter private investment.
The SAZAPIP will focus on three of the six agri-business clusters of the Northern Savannah Ecological Zone (NSEZ)1 and work on enhancing agricultural productivity and production in these areas.
The project will have four Components.
  • Crop Productivity Improvement;
  • Value Chains & Agribusiness Development;
  • Infrastructure Development; and
  • Project Coordination, Management, Monitoring and Evaluation

Project Development Objectives

The objective of this project is to increase agricultural productivity and production to help increase the incomes of farmers, nutrition security and create more jobs in the sector.
